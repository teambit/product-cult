A dashboard panel component that displays a summary of upvotes for a specific item.
It fetches upvote data using the useUpvote hook and presents it in a concise, visually appealing, and informative manner.

## Usage

### Basic Panel

This example demonstrates the basic usage of the `UpvotesDashboardPanel` component.

```jsx
<UpvotesDashboardPanel itemId="product-123" itemType="product" />
```

### Custom Title

This example shows how to customize the title of the `UpvotesDashboardPanel`.

```jsx
<UpvotesDashboardPanel itemId="article-456" itemType="article" title="Article Upvotes" />
```
