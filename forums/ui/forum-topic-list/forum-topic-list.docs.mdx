## Component Usage

The `ForumTopicList` component displays a list of forum topics for a specific forum. Each topic includes the title, author information, and creation date, with a link to the full topic page.

## Basic Example

This example demonstrates the basic usage of the `ForumTopicList` component, displaying a list of forum topics.

```jsx
<ForumTopicList forumId="general-discussion" mockTopics={[
  {
    id: 'topic-1',
    forumId: 'general-discussion',
    title: 'First topic',
    content: 'Content of first topic',
    userId: 'user-1',
    createdAt: new Date().toISOString(),
    updatedAt: new Date().toISOString(),
  },
  {
    id: 'topic-2',
    forumId: 'general-discussion',
    title: 'Second topic',
    content: 'Content of second topic',
    userId: 'user-2',
    createdAt: new Date().toISOString(),
    updatedAt: new Date().toISOString(),
  },
]} />
```
