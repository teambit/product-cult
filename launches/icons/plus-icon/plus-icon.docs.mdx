The `PlusIcon` component displays a plus symbol, typically used to indicate adding, creating, or expanding actions. It can be customized with different sizes, colors, and styles.

## Usage

### Basic Plus Icon

This example shows the default `PlusIcon` with no additional properties.

```jsx
<PlusIcon />
```

### Sized Plus Icons
You can adjust the size of the `PlusIcon` using the `size` property.
```jsx
<PlusIcon size={32} />
```

### Colored Plus Icons
You can change the color of the `PlusIcon` using the `color` property.
```jsx
<PlusIcon color="var(--colors-primary-default, orange)" />
