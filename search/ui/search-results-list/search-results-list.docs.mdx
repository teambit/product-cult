The `SearchResultsList` component is used to display a list of search results. It supports displaying a title, description, and image for each result.

## Basic Usage

This example shows how to use the `SearchResultsList` component with a basic list of search results.

```jsx
<SearchResultsList
  results={[
    {
      id: '1',
      title: 'Test Product',
      description: 'A test product description.',
      type: 'product',
      data: { url: '/test-product' },
    },
  ]}
/>
```

## Empty State

This example demonstrates how the `SearchResultsList` component renders when no results are provided. It displays a message indicating that no results were found.

```jsx
<SearchResultsList results={[]} />
```
