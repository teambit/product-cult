A component for managing search query parameters. It supports keywords, content type filters, sorting options, and pagination settings.

## Get started

Install and use the component in your code:

```ts
const searchQuery = new SearchQuery('example keywords', ['product', 'article'], 'relevance', 'desc', 1, 20);
console.log(searchQuery.toObject());
```

## Basic Search Query

Create a simple search query with keywords.

```ts
const query = new SearchQuery('Bit components');
console.log(query.toObject());
```

## Search with Filters and Pagination

Create a search query with content type filters and pagination.

```ts
const query = new SearchQuery(
  'design',
  ['product', 'component'],
  'name',
  'asc',
  2,
  15
);
console.log(query.toObject());
```
