The `Badge` component is a versatile element for displaying statuses, labels, or categories. It supports different variants, sizes, colors, and an optional icon.

## Usage

### Basic Badge

A simple badge with a label.

```jsx
<Badge label="New" />
```


### Badge Variants

Badges can be displayed in filled, outlined, or ghost variants to match different UI styles.

```jsx
<><Badge label="Filled" variant="filled" color="primary" />
<Badge label="Outlined" variant="outlined" color="secondary" />
<Badge label="Ghost" variant="ghost" color="neutral" /></>
```


### Badge with Icon

Displaying a badge with an icon enhances visual recognition.

```jsx
const StarIcon = ({ className }: { className?: string }) => (

  </svg>

<Badge label="Featured" icon={StarIcon} color="success" />
