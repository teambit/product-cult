The `Logo` component displays a logo, name, and optional slogan, with customizable appearance and linking.

## Basic Usage

Displays the default Product Hunt logo with the brand name.

```jsx
<Logo />
```

## Custom Slogan

Add a slogan to provide context or a tagline for the brand.

```jsx
<Logo slogan="Discover your next favorite thing." />
```

## Minimal Logo

Display only the logo, useful for compact layouts or when the brand is already understood.

```jsx
<Logo minimal={true} />
