## Usage

The `EditUserProfilePage` component allows users to edit their profile information such as name, bio, company, and avatar URL. It fetches the user's profile data and provides a form to update these details.

## Basic Example

This example demonstrates the basic usage of the `EditUserProfilePage` component.

```jsx
<EditUserProfilePage />
```

## Styled Example

This example shows how to apply custom styles to the `EditUserProfilePage` component. You can use CSS classes to modify the appearance of the page and its elements.

```jsx
<EditUserProfilePage className="custom-profile-page-wrapper" />
```
