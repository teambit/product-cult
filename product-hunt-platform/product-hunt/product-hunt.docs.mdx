The Product Cult platform composition. Composes Product Cult platform features into a unified and executable platform.

## Run the platform
To run the Product Cult platform, execute the following commands:

```bash
bit use product-cult
```
Then, start the platform:
```bash
bit run product-cult
```
The platform will start and provide a URL in the command output.
## Compose aspects
To add new features, you can compose additional aspects to the Product Cult platform by modifying the `aspects` array in `product-cult.bit-app.ts`.
```ts
// Example: import { AnotherFeatureAspect } from '@your-scope/another-feature';
export const ProductHunt = HarmonyPlatform.from({
  name: 'product-cult',
  aspects: [
    ProductHuntPlatformAspect,
    UpvotesAspect,
    // your new platform feature.
    // AnotherFeatureAspect,
  ],
});
