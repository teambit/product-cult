The `EditProduct` component provides a form to edit an existing product's details. It fetches the product data, allows modifications, and updates the product information.

## Basic Usage

This example demonstrates the basic usage of the `EditProduct` component. It fetches data and renders an edit form.

```jsx
<EditProduct />
```
## Loading State
This example shows how the `EditProduct` component handles the loading state while fetching product details.
```jsx
<EditProductLoadingState />
```
