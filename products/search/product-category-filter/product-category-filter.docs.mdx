The `ProductCategoryFilter` component allows users to filter products by category. It fetches categories and displays them in a select list.

## Basic Usage

This example demonstrates the basic usage of the `ProductCategoryFilter` component. It displays a list of product categories and allows the user to select one.

```jsx
<MockProvider>
  <div style={{ maxWidth: '350px' }}>
    <ProductCategoryFilter
      mockCategoriesData={[
        new ProductCategory('cat-id-1', 'Tech & Gadgets', 'Latest in technology and electronic gadgets.'),
        new ProductCategory('cat-id-2', 'Software Solutions', 'Applications and platforms for various needs.'),
      ]}
      selectedCategoryId={undefined}
      onCategoryChange={(categoryId) => console.log('Selected category:', categoryId)}
      label="Filter by Category"
      placeholder="All Categories"
    />
  </div>
</MockProvider>
```
## With Initial Selection
This example shows how to pre-select a category in the filter.
```jsx

  <div style={{ maxWidth: '350px' }}>
    <ProductCategoryFilter
      mockCategoriesData={[
        new ProductCategory('cat-id-1', 'Tech & Gadgets', 'Latest in technology and electronic gadgets.'),

      onCategoryChange={(categoryId) => console.log('Selected category:', categoryId)}
      label="Filter by Category"
    />
  </div>
</MockProvider>
```
